# mcp-fdic

FDIC MCP — FDIC BankFind Suite API (free, no auth)

Part of the [Pipeworx](https://pipeworx.io) open MCP gateway.

|------|-------------|

## Quick Start

Add to your MCP client config:

```json
{
  "mcpServers": {
    "fdic": {
      "url": "https://gateway.pipeworx.io/fdic/mcp"
    }
  }
}
```

Or use the CLI:

```bash
npx pipeworx use fdic
```
